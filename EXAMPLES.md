# Примеры использования

## Сценарий 1: Поход с друзьями

### Ситуация
Компания из 4 друзей пошла в поход. Каждый что-то купил, нужно разделить расходы.

### Действия в боте

1. **Создание мероприятия:**
```
/new_event
Название: Поход в горы
Описание: Выходные 15-16 января
```

2. **Добавление участников:**
```
/add_participant
Выбрать: Поход в горы
Тип: Пользователь Telegram
Username: @ivan

/add_participant
Username: @maria

/add_participant
Username: @alex

/add_participant
Username: @olga
```

3. **Добавление расходов:**
```
/add_expense
Мероприятие: Поход в горы
Сумма: 4000
Описание: Палатки напрокат
Категория: Аренда
Разделить: Поровну на всех
```

```
/add_expense
Сумма: 2400
Описание: Продукты для всех
Категория: Еда
Разделить: Поровну на всех
```

```
/add_expense
Сумма: 1200
Описание: Дрова
Разделить: Поровну на всех
```

4. **Расчет:**
```
/calculate
Мероприятие: Поход в горы

Результат:
Всего потрачено: 7600 ₽
На каждого: 1900 ₽

Мария должна:
  → Ивану: 1900 ₽

Алекс должен:
  → Ивану: 1900 ₽

Ольга должна:
  → Ивану: 1900 ₽
```

## Сценарий 2: Вечер с парами

### Ситуация
3 пары пошли в ресторан. У каждой пары общий бюджет.

### Действия в боте

1. **Создание мероприятия:**
```
/new_event
Название: Вечер в ресторане
```

2. **Добавление участников:**
```
/add_participant для всех 6 человек:
@ivan, @maria (пара 1)
@alex, @olga (пара 2)
@sergey, @anna (пара 3)
```

3. **Создание семей:**
```
/create_family
Мероприятие: Вечер в ресторане
Название: Семья Ивановых
Участники: Иван, Мария
Казначей: Иван

/create_family
Название: Семья Петровых
Участники: Алекс, Ольга
Казначей: Алекс

/create_family
Название: Семья Сидоровых
Участники: Сергей, Анна
Казначей: Сергей
```

4. **Добавление расходов:**
```
/add_expense
Сумма: 9000
Описание: Ужин в ресторане
Разделить: Поровну на всех

/add_expense
Сумма: 1500
Описание: Такси до ресторана
Разделить: Поровну на всех
```

5. **Расчет:**
```
/calculate

Результат:
Всего: 10500 ₽
На каждую пару: 3500 ₽

Семья Петровых должна:
  → Семье Ивановых: 3500 ₽

Семья Сидоровых должна:
  → Семье Ивановых: 3500 ₽
```

## Сценарий 3: Долгий отпуск

### Ситуация
Компания уехала в отпуск на неделю. Разные люди платят за разное.

### Действия

1. **Создание мероприятия:**
```
/new_event
Название: Отпуск в Сочи
Описание: 20-27 января
```

2. **День 1 - Аренда жилья:**
```
/add_expense
Сумма: 21000
Описание: Аренда дома на неделю
Категория: Жилье
Разделить: Поровну
```

3. **День 2 - Продукты:**
```
/add_expense
Сумма: 5000
Описание: Продукты на неделю
Категория: Еда
```

4. **День 3 - Развлечения:**
```
/add_expense
Сумма: 8000
Описание: Прокат лодок
Категория: Развлечения
```

5. **Промежуточный расчет:**
```
/calculate
Посмотреть текущие долги
```

6. **В конце отпуска - финальный расчет:**
```
/calculate
Отправить уведомления: Да

Каждому приходит сообщение с его долгами
```

## Сценарий 4: Использование шаблонов семей

### Ситуация
Одна и та же компания часто ходит гулять.

### Первый раз

1. **Создание шаблона семьи:**
```
/create_family_template
Название: Моя семья
Участники: Я, Жена
Казначей: Я
```

2. **Создание мероприятия:**
```
/new_event
Название: Поход в кино
```

3. **Использование шаблона:**
```
/create_family
Использовать шаблон: Моя семья
```

### В следующий раз

```
/new_event
Название: Ужин с друзьями

/create_family
Использовать шаблон: Моя семья
✅ Семья автоматически добавлена!
```

## Сценарий 5: Групповой чат

### Ситуация
Компания общается в групповом чате Telegram.

### Действия

1. **Добавить бота в чат:**
   - Add members → найти бота
   - Сделать админом (опционально)

2. **Создать мероприятие прямо в чате:**
```
/new_event
Название: Встреча друзей
```

3. **Все участники присоединяются:**
```
@ivan: /join_event
@maria: /join_event
@alex: /join_event
```

4. **Кто-то добавляет расход:**
```
@ivan: /add_expense
Сумма: 3000
Описание: Пицца на всех
```

5. **В конце:**
```
/calculate
Отправить уведомления: Да

В чате появляется общий расчет
Каждому в личку приходит его долг
```

## Полезные советы

### Категории расходов

Используйте категории для удобной статистики:
- **Еда**: Продукты, рестораны
- **Транспорт**: Такси, бензин, билеты
- **Жилье**: Аренда, отели
- **Развлечения**: Кино, экскурсии
- **Аренда**: Оборудование, вещи

### Быстрые команды

- `/my_expenses` - посмотреть свои расходы
- `/my_debts` - посмотреть все долги
- `/list_events` - все мероприятия

### Работа с семьями

1. Создайте шаблоны для часто используемых групп
2. Используйте их в новых мероприятиях
3. Один человек из семьи делает все переводы

### Расчеты

- Делайте промежуточные расчеты во время мероприятия
- Финальный расчет с уведомлениями - в конце
- Бот минимизирует количество переводов

## Типичные ошибки

❌ **Неправильно:**
```
Добавить всех как отдельных участников в семейной поездке
```

✅ **Правильно:**
```
Создать семьи, добавить расходы на семьи
```

---

❌ **Неправильно:**
```
Создавать новое мероприятие для каждого дня
```

✅ **Правильно:**
```
Одно мероприятие на всю поездку/неделю
```

---

❌ **Неправильно:**
```
Добавлять мелкие расходы отдельно (кофе за 100₽)
```

✅ **Правильно:**
```
Объединять мелкие расходы: "Разные покупки - 500₽"
```
